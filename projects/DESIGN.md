# DESIGN.md

## 气质与意象

深夜卧室里，手机屏幕亮着暖光，你和对象刚吵完架——空气还有点紧绷，但你知道哄一哄就好了。界面要像一条温暖的微信聊天记录，俏皮但不轻浮，搞笑但有关怀底色。LINE 贴图的可爱感 + 微信私聊的熟悉感。

## 配色方案

- 主背景：奶白暖调 `#FFF8F0`，带微妙暖色
- 聊天气泡（对方）：蜜桃粉 `#FFE4E8`，偏暖偏软
- 聊天气泡（用户）：薄荷绿 `#E0F5EE`，清透舒适
- 主强调色：珊瑚橘 `#FF6B6B`（进度条、按钮、关键信息）
- 加分反馈：翡翠绿 `#22C55E`
- 减分反馈：警示红 `#EF4444`
- 文字主色：深棕灰 `#3D3029`（不用纯黑，保持暖调）
- 文字辅助色：暖灰 `#9B8E82`

## 字体排版

- 中文主体：系统默认中文字体栈
- 标题/数字：圆体感，加粗用 700
- 对话文字：15-16px，行高 1.6
- 选项文字：14-15px，紧凑但可读

## 动效与交互

- 选项飞入：stagger 动画，每项延迟 80ms，从底部弹入
- 气泡出现：轻微 scale + opacity 渐入
- 好感度变化：数字跳动 + 进度条平滑过渡（0.4s ease-out）
- 加分时进度条发绿光脉冲，减分时红色抖动
- 选择后选项收缩消失，对方气泡延迟 300ms 出现
- 游戏结束成功：撒花粒子 + 爱心飘浮
- 游戏结束失败：心碎裂片 + 灰色渐变
- 所有过渡统一使用 `cubic-bezier(0.34, 1.56, 0.64, 1)` 弹性曲线

## 页面结构

- 首页：全屏暖色背景，竖向滚动选择流程（性别→场景→声音→开始），卡片式选择
- 游戏页：顶部状态栏（进度条+轮次）+ 中间聊天气泡区（可滚动）+ 底部选项区
- 结果页：全屏覆盖，居中结果卡片，动画背景

## 设计禁忌

- 不要冷色调（蓝、紫系）作为主色
- 不要严肃商务感，所有文案保持俏皮
- 不要让"减分"体验令人沮丧，奇葩选项要让人笑出来
- 不要纯白色背景，保持暖调
- 不要出现任何真实的社交功能元素（评论、点赞等）
